Para la instalación del programa en linux:

Requiere:

1) Python 3.7 o Superior.
2) Ursina,para Instalarlo: Ejecute en la terminal el comantdo: pip install ursina 
							o pip install https://github.com/pokepetter/ursina/archive/master.zip
							Si presenta algún inconveniente en la instalación visite https://www.ursinaengine.org/ 
							para una guía más completa.
3) Clone o descargue el repo completo del videojuego con " pip install https://github.com/Charliecr94/Proyecto-Final.git.


Si desea finalizar el juego en cualquier momento, sólo debe presionar la tecla "Q"
